# Resume ZIP Parser

An AI-powered Streamlit application that extracts structured information from multiple PDF resumes uploaded as a ZIP file using LangChain and Google Gemini API.

## Features

Upload ZIP file containing multiple PDF resumes

Extract resume text automatically

Parse resumes into structured JSON using AI

Download parsed resume data as a JSON file

Simple and interactive Streamlit UI

## Tech Stack

Python

Streamlit

LangChain

Google Gemini API

PyMuPDF (PDF text extraction)
